# Headshot-Feedback (https://steamcommunity.com/sharedfiles/filedetails/?id=3657747578)
Headshot Feedback is a lightweight DayZ mod that provides audio feedback for headshots, improving hit confirmation without affecting gameplay balance.

When a player lands a headshot on another player, a short sound is played only for the shooter.
No global sounds, no spam, no immersion breaking effects.
